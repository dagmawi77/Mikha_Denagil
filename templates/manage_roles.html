About Role-Based Access Control

What is RBAC?

Role-Based Access Control (RBAC) is a security model that restricts system access based on user roles. Each role is assigned specific permissions (endpoints) that determine which features users with that role can access.

How to Use
  • Create roles for different user types (Admin, Manager, etc.)
  • Click "Manage Access" to assign endpoint permissions
  • Assign roles to users in the User Management section
  • Users will only see menu items they have access to

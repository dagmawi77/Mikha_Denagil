# Website Management - Direct Access Links

If you can't see the "Website Management" menu in the sidebar, you can access these pages directly:

## Direct URLs (after logging in):

1. **Hero Slider Management**
   - URL: `http://localhost:5001/admin/website/hero-slides`
   - Manage homepage slider images

2. **Services Management**
   - URL: `http://localhost:5001/admin/website/services`
   - Add/edit/delete services

3. **Gallery Management**
   - URL: `http://localhost:5001/admin/website/gallery`
   - Upload and manage photos

4. **Donation Info Management**
   - URL: `http://localhost:5001/admin/website/donation`
   - Manage donation methods and payment info

5. **Contact Info Management**
   - URL: `http://localhost:5001/admin/website/contact`
   - Manage contact information

6. **Contact Submissions**
   - URL: `http://localhost:5001/admin/website/contact-submissions`
   - View contact form submissions

7. **View Public Website**
   - URL: `http://localhost:5001/`
   - See the public website

## Menu Location

The "Website Management / የድህረገጽ አስተዳደር" menu should appear in the sidebar:
- **Position**: After "User Management" and before "Member Accounts"
- **Icon**: Screen/Desktop icon
- **Location**: Scroll down in the left sidebar menu

## Troubleshooting

If the menu doesn't appear:
1. **Hard refresh**: Press `Ctrl+F5` (Windows) or `Cmd+Shift+R` (Mac)
2. **Clear browser cache**: Clear cached files and reload
3. **Check sidebar**: Make sure the sidebar is expanded (not collapsed)
4. **Scroll down**: The menu might be below the visible area - scroll down in the sidebar
5. **Use direct URLs**: Access the pages directly using the URLs above
